🔮 NTE Gear & Modules – Stat Module Sets & Build Guide

NTE gear system (Stat Modules). Best modules for DPS, Support & Sub-DPS. Updated: May 22, 2026

🔥 Recommended Stat Modules

DPS Module Set

Focus on ATK% + CRIT Rate + CRIT DMG for main DPS Espers. Pair with matching Arc type for bonus synergy.

Support Module Set

Prioritize HP% + Energy Regen for healers and buffers. Supports benefit more from utility than raw damage.

📍 How to Farm Stat Modules

Anomaly Commissions

Repeatable dungeons that drop Stat Modules. Higher difficulty = better rarity. Unlock more via Tycoon system.

Hunter Exchange (Tycoon Rank 6)

Trade surplus materials for specific Stat Modules. Best way to target-farm BiS gear.

Anomaly Furniture

Place furniture in your apartment for passive stat boosts and module drop rate increases.

🏗️ Module Slot Guide

Slot 1 – Main Arc (Weapon)

Determines primary stat and Arc type. Match with Esper type for 20% bonus.

Slots 2-4 – Stat Modules

Gear slots that stack stats. DPS: CRIT > ATK% > flat ATK. Support: HP% > Energy Regen > DEF%.

Set Bonus

Matching module sets (2pc/4pc) grant bonus effects. Similar to Genshin artifact or WuWa echo sets.
